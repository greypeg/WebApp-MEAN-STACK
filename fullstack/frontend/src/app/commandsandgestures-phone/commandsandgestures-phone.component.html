Commands And Gestures

Voice Commands

When you see this icon on the top right corner of the screen, you can use Voice Commands to make certain actions. A complete list of these commands is described above.

• "Previous Page" : Use this command to navigate to the previous displayed page

• Name of Button : Say the name of each button to navigate to each selection

• "Instant Replay": Use this command to view an instant replay in the TV

• "Next Hashtag" : Use this command to view the next hashtag in the SurroundWall

• "Start The Game": Use this command to initialize the game

Air Gestures

When you see this icon on the top right corner of the screen, you can use Air Gestures to make certain actions. A complete list of these commands is described above.

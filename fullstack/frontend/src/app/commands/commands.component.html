Commands and Gestures

Voice and Commands

When you see this icon on the top right corner of the screen, you can use Voice Commands to make certain actions. A complete list of these commands is described above.

• “Previous Page” : Use this command to navigate to the previous displayed screen.

• Name of Button : Say the name of each button to select it.

• “Instant Replay” : Use this command to view an instant replay on the SurroundWall.

• “Next Hashtag” : Use this command to view the next hashtag in the SurroundWall

• “Start the Game” : Use this command to initialize the game

Air Gestures
